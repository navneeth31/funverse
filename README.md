# Funverse

Funverse is a web-based multiplayer gaming platform where friends can join virtual Gameplay.

**Website:** [https://funverse.netlify.app/](https://funverse.netlify.app/)

## Overview

Funverse offers a simple way to play multiplayer games online. Players can enjoy interactive gameplay directly from their browsers — no downloads or installations required! New rooms and other features rolling out soon.

## Features

- **Real-Time Multiplayer:** Play games together with friends.
- **Simple Interface:** User-friendly, responsive design for quick access and intuitive play.
- **Accessible Anywhere:** Runs on all devices — PC, tablet, and mobile via web browser.
- **Variety of Games:** Enjoy a selection of classic party and social games.
  
## Getting Started

1. Visit [https://funverse.netlify.app/](https://funverse.netlify.app/).
2. Choose a game and enjoy the game.
3. Start playing instantly, no download required!

## Tech Stack

- **Frontend:** React JS
- **Backend:** Node.js, Firebase
- **Real-time Communication:** Not yet Implemented
